# Chatims


This Web Chat App allows users to engage in conversations with strangers, with the option to specify preferences such as gender and age range. Users are matched anonymously for a 7-minute chat. If both express interest with a mutual like, their profiles unlock and they can continue the conversation outside the app.

## Tech Stack

- **Frontend:** Angular

- **Backend:**  Java + Spring Boot

- **Database:**  PostgreSQL

- **REST API:**  Spring Web

- **Realtime:** STOMP over WebSocket

- **Build Tools:**  Maven

- **Testing:**  JUnit / Karma & Jasmine


## Features

🔒 Anonymous one-on-one chat

⏱ 7-minute timed sessions

❤️ Mutual likes unlock user profiles

⚡ Real-time messages and typing indicators

💬 Simple and responsive UI built with Angular

🛡️ Secure authentication powered by Keycloak
